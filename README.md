
## Giới thiệu

Đây là một dự án web đơn giản được xây dựng bằng **HTML, CSS và JavaScript**.
Website mô phỏng một trang web bán sản phẩm với các chức năng cơ bản như hiển thị sản phẩm, điều hướng giữa các trang và tương tác với người dùng.


---

## Ngôn ngữ được dùng

* **HTML5** – xây dựng cấu trúc trang web
* **CSS3** – thiết kế giao diện và bố cục trang
* **JavaScript** – xử lý logic và tương tác người dùng

---

## Cấu trúc dự án

```
project-folder

index.html     # Cấu trúc chính của website
index.css      # File định dạng giao diện
index.js       # File xử lý logic và tương tác
```

---

## Chức năng chính

* Hiển thị các trang nội dung khác nhau (Home, Products, About, ...)
* Điều hướng trang bằng JavaScript
* Xử lý sự kiện người dùng (click, chuyển trang)
* Hiển thị danh sách sản phẩm
* Tương tác giao diện cơ bản với người dùng
* Khả năng lưu trữ cơ bản
* Đăng nhập bằng gmail
* chat bot
* Thanh tìm kiếm
  

---

## Kỹ thuật phần mềm đơn giản áp dụng

Trong dự án này có áp dụng một số kỹ thuật phần mềm cơ bản:


* **Encapsulation trong JavaScript**
  Sử dụng hàm tự thực thi (IIFE) để tránh ô nhiễm biến toàn cục.

* **Event Driven Programming**
  Sử dụng `addEventListener()` để xử lý các sự kiện của người dùng.

* **Function Reusability**
  Các hàm được thiết kế để có thể tái sử dụng trong nhiều tình huống khác nhau.
